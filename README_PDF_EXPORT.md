# PCB Array Optimizer - PDF Export

## Milestone 5: PDF Export - COMPLETED

Professional PDF export functionality has been implemented using ReportLab, allowing users to generate comprehensive optimization reports.

## Features Implemented

### PDF Exporter ([src/export/pdf_exporter.py](src/export/pdf_exporter.py))

The `PDFExporter` class generates multi-page PDF documents containing:

#### 1. Title Page
- Project name
- Generation timestamp
- PCB dimensions summary
- Configuration overview table
  - PCB dimensions
  - Rotation allowances
  - Number of panel templates

#### 2. Configuration Details Page
- **PCB Configuration Table**
  - Width and height (in user's selected units)
  - Area
  - Rotation setting

- **Array Configuration Table**
  - X and Y spacing
  - Top, bottom, left, right rails
  - Array rotation setting

#### 3. Results Summary Table
- Top 10 optimization results in table format
- Columns:
  - Rank (#1, #2, etc.)
  - Utilization percentage
  - Panel size name
  - Array configuration (e.g., "18×20R" for rotated)
  - Arrays on panel (e.g., "1×1")
  - Total PCBs per panel
- Highlighted row for currently selected panel
- Alternating row colors for readability

#### 4. Detailed Panel Layouts (Top 3-4 Results)
For each featured result:
- Rank and panel name heading
- Utilization summary
- **Visual panel diagram** showing:
  - Panel outline (gray)
  - Border keepout areas (dashed gray)
  - Array containers (blue)
  - Individual PCBs (green)
  - Accurate scaling and proportions
- **Specifications table** with:
  - Panel dimensions
  - Border keepouts
  - Array spacing on panel
  - Array configuration and dimensions
  - Total PCBs and utilization metrics

### Features

- **Unit-Aware Display**: All dimensions shown in user's selected unit system (metric or imperial)

- **Color-Coded Elements**: Visual diagrams use consistent color scheme

## Usage

### From GUI

1. Run optimization
2. Select a result
3. Click "Export to PDF" button
4. Choose file location
5. PDF is generated with:
   - All results in summary table
   - Selected panel highlighted
   - Detailed layouts for top results

### Programmatically

```python
from src.models import PCB, ArraySpacing, ArrayRails, Configuration
from src.core import LayoutOptimizer
from src.io.template_manager import TemplateManager
from src.export import PDFExporter

# Create configuration
pcb = PCB(width=25.4, height=20.32, allow_rotation=True)
spacing = ArraySpacing(x_spacing=2.54, y_spacing=2.54)
rails = ArrayRails(top=5.0, bottom=5.0, left=5.0, right=5.0)
panel_sizes = TemplateManager.load_default_templates()

config = Configuration(
    project_name="My PCB Project",
    pcb=pcb,
    array_spacing=spacing,
    array_rails=rails,
    allow_array_rotation=True,
    panel_sizes=panel_sizes
)

# Run optimization
results = LayoutOptimizer.optimize(config, top_n=10)

# Export to PDF
exporter = PDFExporter()
exporter.export_results(
    file_path="output.pdf",
    configuration=config,
    results=results,
    selected_panel=results[0]  # Highlight first result
)
```

## Technical Implementation

### ReportLab Components Used

- **SimpleDocTemplate**: Multi-page document structure
- **Platypus**: High-level layout (Table, Paragraph, Spacer, PageBreak)
- **Drawing**: Vector graphics for panel diagrams
- **Styles**: Custom typography and formatting

### PDF Structure

```
PDFExporter
├── export_results()          # Main entry point
├── _create_title_page()      # Title and summary
├── _create_configuration_summary()  # Config tables
├── _create_results_table()   # Results summary
├── _create_panel_detail()    # Detailed panel page
├── _create_panel_drawing()   # Vector diagram
├── _draw_pcbs_on_drawing()   # PCB rectangles
├── _get_table_style()        # Consistent table styling
└── _add_footer()             # Page numbering
```

### Page Layout

- **Page Size**: Letter (8.5" × 11") by default, configurable to A4
- **Margins**: 0.75" on all sides
- **Typography**:
  - Title: Helvetica 18pt
  - Headings: Helvetica 14pt bold
  - Body: Helvetica 10pt
  - Tables: Helvetica 9pt

### Color Scheme

Consistent with visualization:
- Panel background: `#f0f0f0` (light gray)
- Panel border: `#808080` (gray)
- Usable area: `#c0c0c0` (medium gray, dashed)
- Arrays: `#96B4DC` (blue)
- PCBs: `#649664` (green)
- Table headers: `#4a4a4a` (dark gray)
- Highlighted row: `#ffffcc` (light yellow)
